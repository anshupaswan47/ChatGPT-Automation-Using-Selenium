## Usage
1. Clone the repository to your local machine:
    ```
    git clone https://github.com/anshupaswan47/ChatGPT-Automation-Using-Selenium
    ```
2. Install the required dependencies:
    ```
    pip install -r requirements.txt
    ```
3. Run the script:
    ```
    python script.py
    ```
